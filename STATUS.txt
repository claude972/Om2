Vérifier les services       : sudo supervisorctl status
Redémarrer tous services    : sudo supervisorctl restart all
Redémarrer backend          : sudo supervisorctl restart backend
Redémarrer frontend         : sudo supervisorctl restart frontend

Logs backend                : tail -f /var/log/supervisor/backend.err.log
Logs frontend               : tail -f /var/log/supervisor/frontend.err.log

Repeupler la base           : python /app/scripts/seed_data.py

Test API health             : curl http://localhost:8001/api/health
Test API chantiers          : curl http://localhost:8001/api/chantiers
Test API stats              : curl http://localhost:8001/api/statistiques
